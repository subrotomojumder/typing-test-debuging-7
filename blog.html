Q.1. What is the difference between Local Storage and Session Storage?
Ans:

Local Storage:- local storage URL বা উইন্ডোতে টিকে থাকে । user site close করলেও, user এর experience local storage থেকে যায় । পরে ‍কখনো আগের ঐ site browsing করলে page url / user interface আগের মতো দেখায় । local storage এ সংরক্ষিত data মেয়াদ শেষ হওয়ার তারিখ নেই এবং শুধুমাত্র Java script এর মাধ্যমে browser Local storage এ সংরক্ষিত data clear করা হয়। local storage ‍সংরক্ষন করার ability session storage এ থেকে অনেক বেশি । local storage এ সংরক্ষিত data স্পষ্টভাবে মুছে না যাওয়া পর্যন্ত টিকে থাকে। করা পরিবর্তনগুলি সংরক্ষিত এবং সাইটের বর্তমান এবং ভবিষ্যতের সমস্ত পরিদর্শনের জন্য উপলব্ধ। user এর গোপনীয় নয় এমন কিছু ডেটা ধরে রাখতে, local storage ব্যবহার করতে হয় ।

Session Storage:- session storage local storage এর অনুরূপ । একবার window বন্ধ হয়ে গেলে, session storage মুছে ফেলা হয় । session storage ডেটা শুধুমাত্র সেই window এর tab এ পাওয়া যায় যে tab এ browsing করা হয়েছিল । Session Storage ডেটা স্থায়ী নয় অর্থাৎ উইন্ডো/ট্যাব বন্ধ হয়ে গেলে এটি হারিয়ে যাবে। local storage এর মতো, এটি একই origin policy তে কাজ করে। সুতরাং, সংরক্ষিত ডেটা একই মাধ্যমে পাওয়া যাবে। session storage property প্রতিটি প্রদত্ত মূলের জন্য একটি পৃথক স্টোরেজ এলাকা বজায় রাখে যা page session সময়কালের জন্য বরাদ্দ থাকে, যতক্ষণ পর্যন্ত browser খোলা থাকে, পৃষ্ঠা পুনরায় লোড করা এবং পুনরুদ্ধার করার কাজ session storage করে থাকে।
